The bulk add feature utilizes a tag based syntax that reflects html. While in the long run it is very useful and efficient for adding multiple questions to a homework. Below are the generic formats for each question as well as examples. It is important to follow the format to guarantee that the questions are added properly. Feel free copy and past these examples and adjust them to you own needs.

Example 1: Fill In The Blank
<question type= “fill in the blank”>
<prompt>
What is my name?
</prompt>
<answer>
bob
</answer>
</question>

Example 2: Multiple Choice
<question type= “multiple choice”>
<prompt>
What is my favorite color
</prompt>
<options>
<option>Red</option>
<option>Blue</option>
<option>Green</option>
<option>Pink</option>
</options>
<answer>Green</answer>
</question>

Example 3: Multiple Answer
<question type= “multiple answer”>
<prompt>
What are my favorite colors
</prompt>
<options>
<option>Red</option>
<option>Blue</option>
<option>Green</option>
<option>Pink</option>
</options>
<answers>
<answer>Red</answer>
<answer>Pink</answer>
<answer>Green</answer>
</answers>
</question>

Example 4: Long answer
<question type= “long answer”>
<prompt>What did you do this summer?</prompt>
</question>

Example 5: True False
<question type= “true false”>
<prompt>
Is Centre College in Kentucky?
</prompt>
<answer>true</answer>
</question>

As you can see the syntax is very similar for the question types. For multiple choice/answer, add as many option tags as you need, our examples uses four but you can add more or less. The same applies to the answer tag in the multiple answer. We would recommend making up the bulk add text in a separate document and then pasting it into the application.
